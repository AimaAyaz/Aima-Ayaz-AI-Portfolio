## 🔹 Key Concepts  

### **Neural Networks (Basics)**  
- Inspired by the human brain  
- Neurons perform: inputs → weights → bias → activation → output  
- Deep learning uses multiple layers to learn features automatically  

---

## 🔹 CNN Cheetah Metaphor  

### **Why a Cheetah?**  
- Fast vision  
- Sharp pattern recognition  
- Efficient scanning of the environment  

CNNs behave similarly when processing images.

---

## 🔹 How CNNs Work (Explained Through the Cheetah)  

### **1. Convolution Layers — “Cheetah Eyes”**  
- Detect edges, textures, and patterns  
- Filters slide across the image to create feature maps  
- Early layers → simple patterns  
- Deeper layers → complex shapes  

### **2. Pooling Layers — “Cheetah Focus”**  
- Reduce noise and image size  
- Keep only the most important features  
- Common methods: Max Pooling, Average Pooling  

### **3. Fully Connected Layers — “Cheetah Decisions”**  
- Combine extracted features  
- Make final predictions (e.g., classification)  
- Output depends on task (softmax for multi‑class, sigmoid for binary)  

---

## 🔹 Applications of CNNs  
CNNs are widely used in:  
- Image classification (cats vs dogs)  
- Object detection (self‑driving cars)  
- Medical imaging (tumor detection)  
- Face recognition  
- OCR and quality inspection
